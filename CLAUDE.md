# Stickman Pet - 桌面火柴人桌宠

## 简介
全屏漫游桌面火柴人桌宠，接入 DeepSeek AI 自我进化 + 实时对话，情绪驱动行为。

## 技术栈
- Electron 33 + HTML5 Canvas（全屏透明覆盖 + 点击穿透）
- DeepSeek API（自我进化 + 实时聊天）
- 纯 JS，无框架依赖
- 程序化骨骼动画系统

## 命令
- `npm start` — 启动应用
- `npm run dev` — 开发模式启动

## 工程结构
```
main.js              主进程：全屏窗口 + 点击穿透 + 屏幕感知 + 进化定时器 + IPC
ai-manager.js        AI 核心：进化逻辑 + 实时聊天 + 记忆管理 + 去重引擎
preload.js           IPC 桥接（点击穿透 + 行为规则 + 屏幕感知 + 聊天）
renderer.js          客户端：骨骼动画 + 情绪状态机 + 行为引擎 + 聊天 UI
ai/behaviors.json    行为规则（AI 进化时可自主修改）
ai/rules.md          性格规则（AI 的灵魂）
ai/memory.md         持久化记忆（带去重，30 天压缩）
ai/profile.md        结构化用户画像
ai/evolution-log.md  进化日志
```

## AI 系统架构
- **本地行为引擎**：behaviors.json 规则匹配（应用/标题/时间/空闲/点击）
- **情绪状态机**：5 维情绪（开心/烦躁/无聊/好奇/精力）驱动行为选择和表情
- **实时聊天**：双击火柴人开启对话，DeepSeek API 驱动回应 + 动作
- **自我进化**：每 20 分钟调用 DeepSeek API，感知用户 + 更新行为规则
- **工具系统**：7 感知工具 + 3 自我修改工具（最多 8 轮调用）
- **记忆管理**：双维度去重（Dice + 词频），退出时压缩 + 画像更新

## 动画系统
- 程序化关节角度插值，物理系统（重力/弹跳/碰撞）
- 29 个动作、9 种表情、10 种粒子特效
- 全屏漫游：火柴人在屏幕底部自由行走，其他区域点击穿透
