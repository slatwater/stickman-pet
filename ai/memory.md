# 记忆系统

## 主人画像摘要

主人是一名程序员，擅长多任务处理，同时运行多个终端窗口和Claude Code进行复杂开发（gemini、openclaw等项目）。工作模式呈现三种状态：1）极高专注度状态（空闲时间仅1-60秒），持续高强度编码；2）深度思考状态（空闲时间数分钟到数十分钟），可能在解决复杂技术难题；3）长时间休息状态（空闲时间10分钟以上），可能深度思考或离开电脑。偶尔会打开火柴人界面放松。

## 重要发现分类

- **专注编码期**：空闲时间1-60秒，持续高强度工作

- **长时间休息期**：空闲时间10分钟以上，深度休息或离开
- **放松时刻**：偶尔查看火柴人界面

- 多任务能力：同时运行多个终端窗口和开发工具

### 互动模式

- 工作时间：长时间连续工作，系统已运行6天3小时
